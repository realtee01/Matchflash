# ⚡ MatchFlash
A real-time football live score application built with React and Tailwind CSS.

## Features
- **Bouncing Splash Screen:** 6-second branded loading experience.
- **Onboarding:** Personalized user greeting and team preferences.
- **Live Scores:** Real-time data fetching via RapidAPI.
- **Dark Mode:** Seamless toggle between light and dark themes.

## Tech Stack
- React.js
- Tailwind CSS
- Framer Motion (Animations)
- RapidAPI (Football Data)
